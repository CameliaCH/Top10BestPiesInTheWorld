# Top 10 Pies in the World

A simple webpage created to celebrate Pi Day (π Day)!  
This project showcases 10 popular pies from around the world with images, names, and short descriptions.

## Built With
- HTML5
- CSS3

## 📸 Preview
Displays a collection of pies including:
- Apple Pie
- Pumpkin Pie
- Pecan Pie
- Key Lime Pie
- Cherry Pie
- Chocolate Pie
- Shepherd’s Pie
- Blueberry Pie
- Banoffee Pie
- Meat Pie
